# Progetto fpdf2 - Learning e Testing

Progetto per studiare e testare fpdf2, una libreria Python per generare PDF.

## Setup

```bash
# Crea ambiente virtuale
python3 -m venv venv

# Attiva ambiente virtuale
source venv/bin/activate  # Linux/Mac
# oppure
# venv\Scripts\activate  # Windows

# Installa dipendenze
pip install -r requirements.txt
```

## Struttura

- `examples/` - Esempi base e tutorial
- `practice/` - I tuoi esperimenti personali
- `output/` - PDF generati
- `assets/` - Immagini, font, e altre risorse
